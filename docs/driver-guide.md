## 1. はじめに（アプリのインストール）

スマルトはスマートフォンやタブレットのホーム画面に追加して、アプリのように使えます。

### インストール手順

1. スマートフォンのブラウザでスマルトのURLにアクセスする
2. ログイン画面の上部に **「このアプリをホーム画面に追加してより快適に利用できます」** というバナーが表示される
3. 青い **「インストール」** ボタンをタップ
4. ブラウザの確認ダイアログで **「インストール」** を選択
5. ホーム画面に **「スマルト v3.1」** のアイコンが追加される

> **ポイント**: ホーム画面から起動すると、ブラウザのアドレスバーが表示されず、アプリのように全画面で操作できます。

---

## 2. ログイン

1. アプリを開く（またはブラウザでURLにアクセス）
2. **ログインID** を入力
3. **パスワード** を入力
4. **「ログイン」** ボタンをタップ

### 注意事項

- パスワードを **5回** 間違えると、**15分間** ログインできなくなります
- 一定時間操作しないと自動でログアウトされます。再度ログインしてください
- ログインできない場合は管理者に連絡してください

---

## 3. ダッシュボード（ホーム画面）

ログイン後に表示されるホーム画面です。

### 画面の見方

| エリア | 内容 |
|--------|------|
| **上部バー** | 本日の売上・月間売上・乗車数などの実績 |
| **アラート** | 未完了の点検や点呼など、対応が必要な項目（赤・オレンジで表示） |
| **始業** | 日常点検・乗務前点呼・出庫処理へのショートカット |
| **運行** | 乗車記録・予約カレンダーへのショートカット |
| **終業** | 入庫処理・乗務後点呼・売上金確認へのショートカット |

> **ポイント**: アラートが表示されている場合は、先にそちらを対応してください。

## 4. 1日の業務フロー

毎日の業務は、以下の順番で進めます。

```
 朝（始業）
  │
  ├─ (1) 日常点検 ............... 車両の状態を17項目チェック
  ├─ (2) 乗務前点呼 ............. 健康状態・持ち物を16項目確認
  ├─ (3) 出庫処理 ............... 出発時刻・メーター記録
  │
 日中（運行）
  │
  ├─ (4) 乗車記録 ............... お客様ごとに乗降を記録
  │      ↑ 乗車のたびに繰り返す
  │
 夕方（終業）
  │
  ├─ (5) 入庫処理 ............... 帰着時刻・メーター・経費記録
  ├─ (6) 入庫記録一覧 ........... 過去の入庫記録を確認・修正
  ├─ (7) 乗務後点呼 ............. 業務終了の12項目確認
  └─ (8) 売上金確認 ............. 現金を数えて差異チェック
```

## 5. 日常点検

車両の安全状態を確認する17項目の点検です。

### 操作手順

1. ダッシュボードから **「日常点検」** をタップ
2. **点検者**（自分）と **車両** を選択
3. 各点検項目を確認し、結果を選択する

### 点検項目一覧

**車内での確認（6項目）:**

| # | 項目 | 内容 |
|---|------|------|
| 1 | フットブレーキの踏み代・効き | ブレーキペダルの遊びと制動力 |
| 2 | パーキングブレーキの引き代 | サイドブレーキの引きしろ |
| 3 | エンジンのかかり具合・異音 | エンジン始動時の状態 |
| 4 | エンジンの低速・加速 | アイドリングと加速の状態 |
| 5 | ワイパーのふき取り能力 | ワイパーの拭き取り |
| 6 | ウインドウォッシャー液の噴射状態 | ウォッシャー液の出具合 |

**エンジンルーム（6項目）:**

| # | 項目 | 内容 |
|---|------|------|
| 7 | ブレーキ液量 | リザーブタンクの液面 |
| 8 | 冷却水量 | ラジエーター冷却水の量 |
| 9 | エンジンオイル量 | オイルゲージでの確認 |
| 10 | バッテリー液量 | バッテリーの液面 |
| 11 | ウインドウォッシャー液量 | ウォッシャー液の残量 |
| 12 | ファンベルトの張り・損傷 | ベルトの緩み・ひび割れ |

**灯火・タイヤ（5項目）:**

| # | 項目 | 内容 |
|---|------|------|
| 13 | 灯火類の点灯・点滅 | ヘッドライト・ウインカー等 |
| 14 | レンズの損傷・汚れ | ランプレンズの状態 |
| 15 | タイヤの空気圧 | 目視・触感での確認 |
| 16 | タイヤの亀裂・損傷 | ひび割れ・キズの有無 |
| 17 | タイヤ溝の深さ | 残溝の確認 |

### 結果の入力方法

各項目について、以下のいずれかをタップします。

- **可**（緑）--- 問題なし
- **否**（赤）--- 不良あり
- **省略**（黄）--- 該当しない場合（必須項目は省略不可）

> **時短テクニック**: 全項目に問題がなければ、**「全て可」** ボタンで一括入力できます。

### 保存

- すべての必須項目を入力したら **「登録する」** をタップ
- **「否」** を選んだ項目がある場合は、不良内容を入力してください
- 入力途中で中断しても、30秒ごとに自動で下書き保存されます

| # | 項目 |
|---|------|
| 1 | 健康状態 |
| 2 | 服装 |
| 3 | 履物 |
| 4 | 運行前点検 |

| 9 | 地図 |

## 7. 出庫処理

営業所を出発する際の記録です。

### 操作手順

1. ダッシュボードから **「出庫処理」** をタップ
2. 以下の項目を入力する

| 項目 | 入力内容 | 備考 |
|------|----------|------|
| 運転者 | 自分を選択 | |
| 車両 | 使用する車両を選択 | |
| 出庫日 | 本日の日付 | 自動入力 |
| 出庫時刻 | 出発する時刻 | 現在時刻が自動入力 |
| 天候 | 晴 / 曇 / 雨 / 雪 / 霧 | いずれか選択 |
| 出庫メーター | 車両のオドメーター値（km） | 目視で確認して入力 |

3. **「登録する」** をタップ

> **ポイント**: 車両を選択すると、前回の入庫メーター値が表示されます。オドメーターと照らし合わせて確認してください。

## 8. 乗車記録

お客様を乗せるたびに記録します。1日の中で最も使う画面です。

### 新しい乗車記録の追加

1. ダッシュボードから **「乗車記録」** をタップ
2. **「乗車記録を登録する」** ボタンをタップ（スマホの場合は右下の **「+」** ボタン）
3. 以下の項目を入力する

### 入力項目

| 項目 | 入力内容 | 必須 |
|------|----------|------|
| 運転者 | 自分（自動選択） | ○ |
| 車両 | 使用中の車両（自動選択） | ○ |
| 乗車日 | 本日の日付（自動入力） | ○ |
| 乗車時刻 | お客様が乗車した時刻 | ○ |
| 降車時刻 | お客様が降車した時刻 | |
| 人数 | お客様の人数（1〜20人） | ○ |
| 乗車地 | 乗せた場所 | ○ |
| 降車地 | 降ろした場所 | ○ |
| 経由地 | 途中の経由地（あれば） | |
| 走行距離 | メーターの距離（km） | |
| 運賃 | メーター運賃（円） | ○ |
| 追加料金 | 迎車料・待機料など（円） | |
| 利用券額 | 福祉利用券の金額（円） | |
| 障害者割引 | 障害者割引を適用する場合チェック | |
| 輸送分類 | 目的を選択 | ○ |
| 支払方法 | 現金 / カード / その他 | ○ |
| 備考 | メモ（あれば） | |

### 輸送分類の選び方

| 分類 | 使う場面 |
|------|----------|
| 通院 | 病院への通院 |
| 外出等 | 買い物・用事など |
| 入院 | 病院への入院 |
| 退院 | 病院からの退院 |
| 転院 | 病院間の移動 |
| 施設入所 | 介護施設への入所 |
| その他 | 上記以外 |

### 乗車地・降車地の入力

- 文字を入力すると、**よく使う場所の候補** が自動表示されます
- 候補をタップすると入力が完了します
- 新しい場所は直接入力してください（次回から候補に表示されます）

### 復路（帰りの記録）の作成

お客様を往復で送迎する場合、帰りの記録を簡単に作成できます。

1. 登録済みの往路（行き）の記録にある **「復路作成」** ボタンをタップ
2. 乗車地と降車地が **自動で入れ替わり**、以下が引き継がれます
   - 運転者・車両・日付
   - 人数・運賃・追加料金
   - 輸送分類・支払方法
   - 障害者割引
3. 乗車時刻は現在時刻に自動設定
4. 内容を確認して **保存**

> **ポイント**: 復路の記録には緑色の **「復路」** バッジが表示されます。

### 保存

- 入力が完了したら **「登録する」** をタップ
- 合計金額（運賃 + 追加料金）は自動計算されます

## 9. 入庫処理

1日の業務を終えて営業所に戻った際の記録です。

### 操作手順

1. ダッシュボードから **「入庫処理」** をタップ
2. **未入庫一覧** から自分の車両をタップすると、運転者・車両・出庫メーターが自動入力される
3. 以下の項目を入力する

| 項目 | 入力内容 | 必須 |
|------|----------|------|
| 入庫日 | 本日の日付（自動入力） | ○ |
| 入庫時刻 | 帰着した時刻（自動入力） | ○ |
| 入庫メーター | 車両のオドメーター値（km） | ○ |
| 燃料代 | 給油した場合の金額（円） | |
| 高速代 | 高速道路を使った場合の金額（円） | |
| その他費用 | その他の経費（円） | |
| 休憩場所 | 休憩した場所 | |
| 休憩開始 | 休憩の開始時刻 | |
| 休憩終了 | 休憩の終了時刻 | |
| 備考・特記事項 | 連絡事項など | |

4. **「登録する」** をタップ

> **ポイント**: 走行距離（入庫メーター − 出庫メーター）は自動計算されます。500kmを超える場合は確認メッセージが表示されます。

## 10. 入庫記録一覧

過去の入庫記録を確認・修正するための画面です。

### 操作手順

1. ダッシュボードのメニューまたは入庫処理ページから **「入庫記録一覧」** をタップ
2. **日付** を選択して検索（初期値は当日）
3. 車両で絞り込むことも可能

### 表示される情報

各レコードはカード形式で表示され、以下の情報が確認できます。

| 項目 | 内容 |
|------|------|
| 入庫時刻 | 帰着した時刻 |
| 運転者・車両 | 担当ドライバーと車両番号 |
| 出庫メーター / 入庫メーター | 出発時と帰着時のオドメーター値 |
| 走行距離 | 自動計算された距離（赤字で強調） |
| 燃料代・高速代・その他費用 | 当日の経費内訳 |
| 備考 | 特記事項 |

### 日次集計

画面右側（スマホでは下部）に当日の集計が表示されます。

- **入庫回数** --- その日に何回入庫したか
- **総走行距離** --- 当日の合計距離（km）
- **総費用** --- 燃料代・高速代などの合計（円）

### 記録の修正

入力ミスに気づいた場合、記録を修正できます。

1. 修正したいレコードの **「修正」** ボタンをタップ
2. 修正画面（モーダル）が開く
3. 修正できる項目：入庫日・入庫時刻・入庫メーター・燃料代・高速代・その他費用・備考
4. **修正理由** を入力（必須）
5. **「保存」** をタップ

> **ポイント**: 修正された記録には黄色の **「修正済み」** バッジが表示されます。修正理由と修正者名も記録されます。

### 注意事項

- 一般ドライバーは **自分の記録のみ** 表示されます
- 入庫メーターが出庫メーターより小さい場合は警告が表示されます
- 走行距離が500kmを超える場合は確認メッセージが表示されます

## 11. 乗務後点呼

業務終了後の最終確認です。

### 操作手順

1. ダッシュボードから **「乗務後点呼」** をタップ
2. **運転者**（自分）を選択
3. **点呼執行者**（立会人）を選択
4. 12項目のチェックボックスを確認してチェック

### 確認項目（12項目）

| # | 項目 |
|---|------|
| 1 | 乗務記録の記載は完了しているか |
| 2 | 車両に異常・損傷はないか |
| 3 | 健康状態に異常はないか |
| 4 | 疲労・睡眠不足はないか |
| 5 | 酒気・薬物の影響はないか |
| 6 | 事故・違反の発生はないか |
| 7 | 業務用品は適切に返却されているか |
| 8 | 業務報告は完了しているか |
| 9 | 車内の忘れ物はないか |
| 10 | 事故・違反の最終確認 |
| 11 | 予定路線での運行は適切だったか |
| 12 | 乗客に関する特記事項はないか |

### アルコール検査

- **アルコール値** を入力（通常 0.000）

### 保存

- すべて確認したら **「登録する」** をタップ

## 12. 売上金確認・現金カウント

1日の売上金を確認し、現金を数えて差異がないかチェックします。

### 操作手順

1. ダッシュボードから **「売上金確認」** をタップ
2. 本日の売上サマリー（乗車回数・売上合計・現金/カード内訳）を確認
3. 手元の現金を金種ごとに数えて入力する

### 現金カウント入力

金種ごとに枚数を入力します。**+** / **-** ボタンまたは直接数値入力ができます。

| 金種 | つり銭基本在庫 |
|------|--------------|
| 1万円札 | 0枚 |
| 5千円札 | 1枚 |
| 千円札 | 10枚 |
| 500円玉 | 3枚 |
| 100円玉 | 11枚 |
| 50円玉 | 5枚 |
| 10円玉 | 15枚 |

> **つり銭基本在庫**: 合計 **18,000円** が標準のつり銭として常時保持する金額です。

### 計算の仕組み

```
カウント合計 = 各金種の枚数 × 金額 の合計
入金額 = カウント合計 − 18,000円（つり銭基本在庫）
差異 = 入金額 − 現金売上（乗車記録から自動集計）
```

### 差異の見方

| 表示 | 意味 |
|------|------|
| **青（0）** | ぴったり一致 |
| **緑（+）** | 現金が多い（おつりの渡し忘れ等） |
| **赤（-）** | 現金が足りない（おつりの渡し過ぎ等） |

- 差異がある場合は **メモ欄** に理由を記入してください
- **「現金カウント保存」** をタップして完了

## 13. よくある質問（FAQ）

### Q. 入力を間違えた場合は？

**当日の記録**: 各画面の **「編集」** ボタンから修正できます。

**過去の記録**: 管理者に連絡してください。管理者のみが修正理由を入力して編集できます。

---

### Q. 途中で画面を閉じてしまった場合は？

日常点検は30秒ごとに自動で下書き保存されます。再度開くと **「下書きデータがあります」** と表示されるので、**「復元」** をタップしてください。

---

### Q. ネットがつながらない場所では？

スマルトはオフラインでも一部の機能が使えます。ネットワークに接続された際にデータが自動で同期されます。

---

### Q. セッションが切れてログイン画面に戻った場合は？

一定時間操作しないと自動ログアウトされます。再度ログインしてください。入力中のデータは下書き保存されている場合があります。

---

### Q. アプリの動作がおかしい場合は？

1. アプリを閉じて再度開く
2. それでも直らない場合はブラウザのキャッシュをクリアする
3. 改善しない場合は管理者に連絡してください

---

> **困ったときは管理者に連絡してください。**
