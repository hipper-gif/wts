2. スマホにアプリを入れる

スマルトはスマホのブラウザからアプリのように使えます。
ホーム画面に追加すると、毎回URLを入力する手間がなくなります。

iPhone（Safari）の場合

1

Safariでシステムにアクセス

上のURLをSafariで開きます。
Chrome等ではなく Safari をお使いください。

2

共有ボタンをタップ

画面下の 四角に上矢印のマーク（共有ボタン）をタップします。

3

「ホーム画面に追加」をタップ

メニューを下にスクロールして「ホーム画面に追加」を探してタップします。

4

「追加」をタップ

名前はそのまま「スマルト」でOKです。右上の「追加」をタップします。
ホーム画面に緑色のアイコンが追加されます。

Android（Chrome）の場合

1

ChromeでシステムURLを開く

上のURLをChromeで開きます。

2

「インストール」バナーが出たらタップ

画面に「ホーム画面に追加」のバナーが出ます。
出ない場合は、右上の 3つの点（メニュー） →「アプリをインストール」をタップします。

次回からはホーム画面の「スマルト」アイコンをタップするだけでOKです。

4. 最初にやること（初日チェックリスト）

以下の項目は初回のみです。2日目からは「3. 1日の業務」だけでOKです。

鈴木さん（管理者）が確認すること

  • アプリをスマホのホーム画面に追加する（2ページ目を参照）
  • ログインできることを確認する
  • 会社情報を確認する（メニュー → マスタ管理 → 会社情報設定）
  • 車両情報に車検有効期限を入力する（メニュー → マスタ管理 → 車両管理）
  • よく行く病院・施設を場所マスタに登録する（メニュー → マスタ管理 → 場所マスタ）
  • 許可証・保険証券のPDFを登録する（メニュー → マスタ管理 → 書類管理）
  • 1日の業務をテスト実行してみる（日常点検 → 乗務前点呼 → 出庫 → 乗車記録 → 入庫 → 乗務後点呼 → 売上確認）

松尾さん（点呼者）が確認すること

  • アプリをスマホのホーム画面に追加する
  • ログインできることを確認する
  • 乗務前点呼の画面で、運転者・点呼者が正しく選べることを確認する
  • 乗務後点呼の画面を開いてみる
テストデータは後から消せます。テストで入力した記録は管理者メニューの「データ管理」から削除できるので、気軽に試してください。

6. 法定帳票とスマルトでの対応

介護タクシー事業者に必要な帳票は、すべてスマルトで記録・出力できます。

法定帳票 スマルトの画面 保管期間
点呼記録簿 乗務前点呼 / 乗務後点呼 3年間
乗務記録（運転日報） 乗車記録 + 出庫・入庫処理 3年間
日常点検記録 日常点検 1年間
運転者台帳 ユーザー管理 / 乗務員台帳 退職後3年間
事故記録 事故管理 3年間
苦情処理記録 苦情処理記録 保管義務あり
指導監督記録 指導監督記録 3年間
定期点検記録 定期点検 1年間
事業報告書（第4号様式） 陸運局報告 提出後控え保管
スマルトに入力していれば、監査時に「陸運局報告」画面からまとめてPDF出力できます。
毎日こつこつ入力しておくことが、監査対策の一番の近道です。

7. よくある質問

Q. パスワードを忘れました
管理者（鈴木さん）がユーザー管理画面からパスワードをリセットできます。
管理者自身のパスワードを忘れた場合は、Smileyまでご連絡ください。
Q. 間違えて入力してしまいました
当日中なら各画面の「編集」ボタンから修正できます。
翌日以降は自動でロックされますが、管理者権限で理由を入力して解除できます。
Q. 往復の送迎を記録したい
行きの乗車記録を登録した後、「復路作成」ボタンをタップすると、
乗車地と降車地が入れ替わった状態で新しい記録が作られます。
Q. 圏外でも使えますか？
基本的な画面表示はキャッシュされていますが、データの保存にはインターネット接続が必要です。
送迎中はメモしておいて、電波のあるところで入力してください。
Q. 複数のスマホやPCで使えますか？
はい。同じIDでログインすれば、スマホ・タブレット・PCどれからでも使えます。
データはすべてサーバーに保存されるので、端末を変えても記録は消えません。
Q. 陸運局の監査で紙の帳票を見せる必要がある場合は？
「陸運局報告」画面から各帳票のPDFを出力・印刷できます。
日常的な入力をしていれば、監査前に慌てる必要はありません。
